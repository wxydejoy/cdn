# Blobcat 表情包

## 来源

表情包来自 [Blob.cat](https://blob.cat) 实例。

Emojis copied from [Blob.cat](https://blob.cat).

## 在 [waline](https://waline.js.org/)评论系统中使用

将一下链接添加到 emoji 选项中

```markdown
https://cdn.jsdelivr.net/gh/norevi/waline-blobcatemojis@1.0/blobs
```
